Probabilidade

Lembretes

Alguns lembretes sobre os principais conceitos de probabilidade podem ser vistos abaixo:

  • Espaço amostral é o conjunto de todos os resultados possíveis de um evento aleatório. Conhecido também como conjunto universo.
  • O ponto amostral é um elemento que pertence ao espaço amostral, ou seja, um entre os vários resultados possíveis do experimento aleatório. Exemplo: quando jogamos um dado, os números 1 a 6 são os pontos amostrais, são os números possíveis de cair ao jogar o dado.
  • Cálculo de probabilidade: 

    \[\mathbf{P(A)} = \frac{\mathbf{n(A)}}{\mathbf{n(\Omega)}}\] 

  • P(A) → probabilidade do evento A

  • n(A) → número de elementos no conjunto A

  • n(Ω) → número de elementos no conjunto

    Observações:

    A probabilidade pode ser representada como fração, como porcentagem ou como número decimal.
    A probabilidade é sempre um número decimal entre 0 e 1, ou uma porcentagem entre 0% e 100%.
    Se P(A) = 0 então A é um evento impossível.
    Se P(A) = 1 então A é um evento certo.
